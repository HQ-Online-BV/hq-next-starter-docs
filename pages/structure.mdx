# Project structuur

Een headless website bij HQ Online bestaat uit 3 verschillende onderdelen:

1. Een Wordpress backend (gebaseerd op de [starter](https://headless.hq-beheer.nl))
1. Een Next.js Front-end (gebaseerd op de [starter](https://github.com/HQ-Online-BV/hq-next-starter-test-2))
1. Hosting via [vercel](https://vercel.com/hq-online-team)

Deze documentatie focust vooral op de Next.js front-end, maar er zullen stukken van de andere onderdelen in voorkomen.
